## DOM方法
```
  getNextElement
    下一个兄弟节点

  getPrevELement
    上一个节点

  getNextOne
    下一个节点

  getNextAll
    所有的下一个节点，数组

  insertAfter
    元素后面添加

  prepend
    前面添加

  getLastChild
    最后一个

  text
    获取文本值，包括空格

  getElementById

  getElementByClassName

  addClass
  removeClass

  getElementChildren
    指定标签名的节点

  getIndex
    获取索引值

获取设置 css属性
  getCss
  setCss


  getPosition
    相对于屏幕的距离

  siblings
    所有兄弟节点，不管前后关系

```
