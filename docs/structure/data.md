# 数据结构
* 数据结构是一个复杂程序的核心
* 知道咋什么场景中使用哪种结构，这本身就是一个艺术
* 数据结构与时间复杂度直接相关，因为选错了数据结构，可以让程序陷入停顿
* 练习，练习，再练习，没有大量的练习，是无法成为专家的



### 数据结构方面的书籍
* 《算法导论》
* 《重构》
* 《代码整洁之道》
* 《程序员修炼之道》
